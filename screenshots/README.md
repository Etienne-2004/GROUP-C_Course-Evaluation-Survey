## Instructions for Capturing Screenshots

1. **Login Page**: Navigate to `http://localhost:8080/login`
2. **Admin Dashboard**: Login as `rp_admin/password` → Navigate to `/admin/dashboard`
3. **User Management**: Admin dashboard → Click "Users" or navigate to `/admin/users`
4. **Survey Creation**: Login as initiator → Navigate to `/initiator/dashboard` → Click "Create Survey"
5. **Analytics**: Any dashboard with survey results and charts
6. **Course Management**: Admin dashboard → Click "Courses" or navigate to `/admin/courses`
7. **Student View**: Login as student → Navigate to `/student/dashboard`

## Tips for Quality Screenshots

- **Use browser zoom**: Capture full page without scrolling
- **Clear browser cache**: Ensure latest UI elements are visible
- **Highlight interactions**: Show hover states and active elements
- **Consistent sizing**: Use 1920x1080 resolution for consistency
- **Include context**: Show browser window for reference
- **Good lighting**: Ensure text is readable and professional appearance
